### React + TypeScript + Vite + Tailwindcss

## Libs
   # [ViteJS](https://vitejs.dev/guide/)
   # [Tailwindcss](https://tailwindcss.com/docs/guides/vite)

## Commands to run
#### in /client run to install dependency
   
   ```bash
    pnpm: pnpm install
    yarn: yarn install
    npm: npm install
   ```

#### Run this project
   ```bash
      pnpm: pnpm dev
      yarn: yarn dev
      npm: npm run dev
   ```
